## 📌 О проекте

Модель компьютерного зрения для **оценки глубины** по аэрофотоснимкам с дрона.  
Модель предсказывает карту глубины (расстояние от камеры до объектов) на основе RGB-изображения.

| Характеристика | Значение |
|----------------|----------|
| **Архитектура** | U-Net с skip connections |
| **Тип задачи** | Регрессия (попиксельное предсказание) |
| **Датасет** | FlyAwareV2 (реальные снимки с дрона) |
| **Размер изображения** | 256×256 пикселей |

---
